四月 12, 2019
  • python

定义

装饰器的本质是函数，功能是用来装饰，装饰的对象是其他的函数，就是为其他函数添加附加的功能。

原则

新增一个功能时不应修改函数的源代码；
新增一个功能时不应修改函数的调用方式；
装饰器对于被装饰函数是透明的；

知识储备

  • 函数即变量

  • 高阶函数
    满足两个条件之一：
    a：把一个函数名当作实参传给另一个函数的；

b：返回值中包含函数名的；

  • 嵌套函数
    高阶函数 + 嵌套函数 = 装饰器
