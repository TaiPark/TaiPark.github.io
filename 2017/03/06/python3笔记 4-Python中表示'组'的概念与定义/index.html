4-6 dict 字典

key-value，最多的操作是通过 key 找到 value一个字典可以由很多 key/value 构成，是无序的，和集合 set 行为很像基本定义方式：{key1:value1,key2:value2,....}

value 可以取任意数据类型 (甚至还能是一个字典)key 必须使不可变的类型：int、str 都是属于不可变的类型
